Ruby on Rails

Ruby on Rails, or Rails, is a server-side web application framework written in Ruby under the MIT License. Rails is a model–view–controller (MVC) framework, providing default structures for a database, a web service, and web pages. It encourages and facilitates the use of web standards such as JSON or XML for data transfer and HTML, CSS and JavaScript for user interfacing. In addition to MVC, Rails emphasizes the use of other well-known software engineering patterns and paradigms, including convention over configuration (CoC), don't repeat yourself (DRY), and the active record pattern. (From Wikipedia)

#installation

Click There to install the Ruby. Not Need to follow the Picture. Just look at the most special one and you can install it.

Note: Make sure that the application is installed on the correct System Path.

After install the exe application, Open your cmd and typing gem imstall rails. Give him some time to install. Make sure correct the Internal.

After installation finish, type rails -v then it will show the ruby version.

Update the ruby version, type gem update rails
